# Tic Tac Toe

Bu proje, klasik **X-O (Tic Tac Toe)** oyununun Python kullanılarak Jupyter Notebook ortamında geliştirilmiş bir uygulamasıdır.

## İçerik
- Oyun tahtasının oluşturulması  
- Oyuncu hamlelerinin alınması  
- Kazanma ve beraberlik kontrolü  
- Oyun akışının yönetimi  

## Gereksinimler
- Python 3.x  
- Jupyter Notebook  

## Kullanım
1. `TicTacToe.ipynb` dosyasını Jupyter Notebook üzerinde açın.  
2. Hücreleri sırasıyla çalıştırın.  
3. Hücre çıktıları üzerinden oyunu oynayın.

## Amaç
Python ile temel programlama yapılarının (koşullar, döngüler, fonksiyonlar) oyun mantığı üzerinden uygulanmasını göstermek.

## Notlar
- Kod eğitim ve örnekleme amaçlıdır.  
- İki oyunculu (insan–insan) yapıdadır.  
- İstenirse AI oyuncu veya grafik arayüz eklenerek geliştirilebilir.
